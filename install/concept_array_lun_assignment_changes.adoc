:icons: font

[role="lead"]
Sie können die Zuordnung einer Array-LUN _spare_ von einem ONTAP-System zu einem anderen ändern. Möglicherweise möchten Sie die Eigentümerschaft für den Lastausgleich über Nodes ändern.
